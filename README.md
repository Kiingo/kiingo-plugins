# Kiingo Plugins

A collection of specialized plugins for AI-powered business operations.

## Available Plugins

### Vistage Chair Toolkit

Complete toolkit for peer advisory group facilitators including Vistage Chairs, EO moderators, YPO forum officers, and peer advisory group leaders.

**Features:**
- **Meeting Architect**: Design complete meetings with room-undercurrent maps, agenda triage, Chair run-of-show, exercises, live pivot scripts, energy management, and member materials
- **Deck Builder**: Rebuild or create on-brand Vistage Chair PowerPoint meeting decks with Vistage logos, Montserrat fonts, custom icons, consistent headers/footers, and native issue-processing funnel slides
- **Issue Processing**: Advanced 5-step issue processing with issue triage, root cause analysis, and accountability systems
- **Exercise Lab**: Generate decision-forcing exercises with facilitation scripts, worksheets, and selection tags for trust, time, guest-safety, and risk
- **Curriculum Toolkit**: Design curriculum as retention and group-health strategy with 12-month red-thread arcs, veteran stretch, renewal-risk planning, check-in diagnostics, speaker integration, and stagnation repair
- **One-to-One Coach**: Prepare private monthly member coaching sessions with member pattern diagnosis, renewal-risk radar, hard conversation scripts, and group-integration judgment
- **Member Onboarding**: Build first-90-day onboarding plans with room integration, old-guard resistance handling, buddy design, first-value creation, and first issue timing
- **Guest Day Converter**: Plan authentic guest visits and post-meeting conversion debriefs
- **Accountability System**: Track commitments, commitment debt, accountability partners, non-shaming resets, follow-through culture, and quarterly progress
- **Group Composition**: Analyze seat strategy, room fit, industry conflicts, member transitions, and second-group readiness
- **Market Map Builder**: Define the room, build a seat thesis, choose target segments, map Chair credibility lanes, build a company universe, score fit/timing/access, and map referral/trust paths
- **Chair Authority Builder**: Build Chair authority through positioning, pattern-based content, speaking topics, newsletters, advisor briefings, and reputation gravity
- **Referral Source Briefs**: Prepare source-specific briefs for members, alumni, advisors, past guests, and connectors before asking for introductions
- **Referral Activation Engine**: Build a trust-preserving referral system with member/advisor/alumni source maps, specific asks, intro language, close-loop cadence, and referral tracking
- **Public Trigger Monitor**: Create public-web trigger monitoring and weekly digests for expansion, leadership, ownership, hiring, award, succession, and event signals
- **Stalled Prospect Reviver**: Diagnose ghosted, deferred, post-guest, "not now," and dormant prospects; choose revive, nurture, park, or pass; and draft relationship-safe manual reactivation messages
- **Prospect Signals**: Produce public-web recruiting signal fiches with trigger events, durable ownership facts, and room-fit questions without LinkedIn automation
- **Prospect Intel**: Research prospects with company analysis, leadership profile, Vistage fit, room-fit overlay, and conversation strategy
- **Recruitment Playbook**: Build recruiting pipeline, referral rhythms, weekly Chair growth operating system, and 30/60/90-day sprints
- **Outreach Campaign**: Create source-specific outreach sequences for referrals, events, cold LinkedIn, past guests, and stalled prospects
- **Objection Battlecards**: Handle objections with psychology-backed responses and Chair story banks
- **Sales Coach**: Practice and debrief prospect conversations with Chair selling-posture diagnosis, fit-protection scoring, objection decoding, and next-step discipline
- **Humanizer**: Remove AI-generated writing patterns for natural, human-sounding text

**Commands:**
- `/meeting-prep [theme]` - Quick-start a complete meeting design
- `/exercise [topic]` - Generate decision-forcing exercises
- `/check-in [context]` - Get curated check-in and icebreaker questions

See [vistage-chair-toolkit/README.md](vistage-chair-toolkit/README.md) for full documentation.

## Installation

Each plugin is available as a `.plugin` file that can be installed directly.

## Development

To add or modify a plugin:
1. Create a directory with your plugin name
2. Include `SKILL.md` files for each skill
3. Include command files in a `commands/` directory
4. Add a `README.md` with plugin overview
5. Create a `.plugin` binary for distribution
